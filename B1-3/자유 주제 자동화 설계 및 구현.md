# 자유 주제 자동화 설계 및 구현

- 개요

---

## 1. 자동화할 반복 업무 정의

**업무명**: 시험·과제 일정 자동 분류 및 알림

**문제 상황**:

Notion에 시험이나 과제 일정을 추가할 때마다 유형별로 따로 정리하고 중요한 일정을 놓치지 않기 위해 직접 확인해야 하는 번거로움이 있었다. 특히 시험 준비와 과제 마감이 겹치는 시기에는 일정 관리가 더욱 복잡해지는 문제가 있었다.

**자동화 목표**:

Notion DB에 새 항목을 추가하면 시험/과제 여부를 자동으로 분류하고, Gmail로 알림 메일을 발송하며 구글 시트에 자동 기록되도록 한다.

---

## 2. 도구 선정 및 선정 이유

**선정 도구**: Make (make.com)

**선정 이유**:

- Router 기능을 통해 하나의 시나리오에서 시험/과제 두 가지 경로를 동시에 처리할 수 있어 관리가 편리함
- Notion, Gmail, Google Sheets 연동이 모두 가능해 별도의 추가 도구 없이 구현 가능
- 시각적 노드 기반 UI로 전체 흐름을 한눈에 파악하기 쉬움
- 무료 플랜(1,000 Ops/월) 내에서 구현 가능

---

## 3. 워크플로우 흐름 설명

```
[Trigger]
Notion 시험·과제 관리 DB에 새 항목 추가
        ↓
[Router - 조건 분기]
선택(유형) = "시험"?
        ── YES ──→ [Action 1] Gmail "시험 알림" 메일 발송
                   [Action 2] 구글 시트 "시험" 탭에 기록
        └── NO  ──→ [Action 1] Gmail "과제 알림" 메일 발송
                    [Action 2] 구글 시트 "과제" 탭에 기록
```

**단계별 설명**:

1. **Trigger** — Notion Watch Data Source Items
    - 시험·과제 관리 DB에 새 항목이 추가되면 자동으로 감지
2. **Router** — 조건 분기
    - `Properties Value → 선택 → Name` 값이 `시험`이면 시험 경로로 분기
    - 그 외(과제)는 과제 경로로 분기
3. **Action 1 (시험)** — Gmail Send an Email
    - 제목: `[시험 알림] {할 일 이름}`
    - 내용: 과목명, 시험 날짜 포함
4. **Action 2 (시험)** — Google Sheets Add a Row
    - 구글 시트 `시험` 탭에 이름, 과목, 날짜 자동 기록
5. **Action 1 (과제)** — Gmail Send an Email
    - 제목: `[과제 알림] {할 일 이름}`
    - 내용: 과목명, 마감일 포함
6. **Action 2 (과제)** — Google Sheets Add a Row
    - 구글 시트 `과제` 탭에 이름, 과목, 날짜 자동 기록
